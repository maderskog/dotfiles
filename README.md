
Description
========

This is my personal collection of dotfiles.

This far, it covers:
* vim
* zsh
* git

Planning to add:
* OS X
* bash
* iterm

Still thinking about:
* ssh


Installation
============

In your home directory:

    $ git clone https://github.com/maderskog/dotfiles.git
    $ cd dotfiles
    $ ./setup
